User Classification Criteria

Understanding User Categories and Recapture Conditions

Classification Criteria Metric Type
New User User's `First Appearance` or `Adoption Date` is less than 90 days from the `reference_date` (the max report date). Single (Time-based)
Power User `Adjusted Consistency (%)` is greater than 75% AND `Usage Complexity` is greater than 10. Multiple
Consistent User `Adjusted Consistency (%)` is greater than 75% (and not classified as "Power User"). Single
Coaching Opportunity `Adjusted Consistency (%)` is greater than 25% (and not classified as "Power User" or "Consistent User"). Single
License Recapture Condition 1: `Overall Recency` (last seen activity) is more than 90 days from the `reference_date`.
OR
Condition 2: If `Adjusted Consistency (%)` is 25% or less (and not classified as "New User"). 		Single (Time-based or Consistency-based)
